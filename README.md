# Computationally guided platform for microbial synthesis of non-natural antinematode phenylpropanoid

## Description

This repository contains the retrobiosynthesis tools integrated with DeepRFC (Deep learning-based Reaction Feasibility Classifier) to evaluate synthetic pathways.
Note: This source code was developed in Linux, and has been tested in Ubuntu 16.04 with Python 3.6. Anaconda was used as a package manager.

## Procedure

1. Clone the repository

        git clone https://github.com/kaistsystemsbiology/mbel_retrobiosynthesis.git

2. Create and activate a conda environment

        conda env create -f environment.yml
        conda activate deeprfc
        
3. If pip error occurs due to Pytorch version,  
   -Remove followings from 'environment.yml'.

        Torch==1.2.0+cu92
        Torchvision==0.4.0+cu02

    -Install an appropriate version of Pytorch, according to your computing envrionment.  
      Please refer to https://pytorch.org/get-started/locally/

##Example

- Run python script

        python run_single_step.py -i 'InChI=1S/C6H12O6/c7-1-2-3(8)4(9)5(10)6(11)12-2/h2-11H,1H2/t2-,3-,4+,5-,6?/m1/s1' -o './output' -n 1
